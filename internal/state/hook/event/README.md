# Events

## Core events that are hookable
- damage_unit x
- heal_unit x
- move_unit x
- attack_unit x
- kill_unit x
- place_unit
- play_card
- draw_card
- burn_card
- sack_card
- refresh_mana
- use_mana
- gain_mana
- lose_mana
- end_turn

## Non hookable events
- damage_units x
- heal_units
- add_trait_to_unit
- remove_trait_from_unit
- swap_unit
